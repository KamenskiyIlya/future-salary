# Сбор статистики по вакансиям программистов

Данный скрипт позволяет собрать статистику по вакансиям связанными с 10ю самыми популярными языками программирования с таких сайтов как [Head Hunter](https://hh.ru/) и [Super Job](https://www.superjob.ru/).

## Что необходимо подготовить

Скачиваете репозиторий.

Для работы скрипта будет необходима установка Python 3. После чего нужно подгрузить необходимые пакеты для нормальной работы скрипта с помощью pip. Для этого введите в командной строке команду:

```
pip install -r requirements.txt
```

Для того чтобы Super job предоставлял Вам данные о вакансиях, необходимо создать приложение на их сайте и получить ключ(токен) от него. Сначала необходимо зарегистрироваться на сайте и войти - [регистрация](https://api.superjob.ru/register). После того как Вы зарегистрировались, заходите во вкладку "Ваше приложение" и создаете свое приложение:
- имя любой
- краткое описание, что будете делать(в нашем случае получать информацию о вакансиях)
- сайт можно указать любой(например `https://localhost/`)
- Ваши данные: почта и имя
- Остальное не обязательно указывать

![sj_app_info](https://github.com/user-attachments/assets/24f0086f-5599-4132-becf-e15275a5d825)


После создания, Вам отобразится Secret key(токен), не сообщайте его никому. В репозитории, который Вы скачали необходимо создать файл .env переменного окружения и записать в него Ваш токен в следующем формате:
```
SJ_TOKEN='Ваш токен'
```

## Как пользоваться

После того, как сделано все из предыдущего пункта, открываем командную строку, указываем путь до скачанного репозитория следующей командой:
```
cd *путь до директории*
```
И запускаем скрипт следующей командой:
```
python salary_statistics.py
```
Скрипту нужно некоторое время, чтобы получить ответ от сайтов Head Hunter и Super Job. В результате скрипт выведет Вам 2 таблицы со статистикой вакансий по каждому сайту отдельно:

![Снимок экрана 2026-01-15 214540 (2)](https://github.com/user-attachments/assets/4688d742-7895-4d93-97fc-f1a1e9a26b59)

Немного подробнее о информации выводимой в таблице:
- Название языка программирования
- Кол-во найденных вакансий с указанным языком программирования
- Вакансий обработано - сколько вакансий учтено при подсчете средней зарплаты
- Средняя зарплата - средняя ЗП среди вакансий с указанной ЗП. Вакансии, в которых не указана ЗП, не учитываются при вычислении средней ЗП

> Если в столбце "средняя зарплата" Вы увидите `None`, это говорит о том, что по критериям отбора не найдена ни одна вакансия где указана ЗП.
